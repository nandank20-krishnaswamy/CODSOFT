# CODSOFT
1. Simple To‑Do List Application
Command‑line based task manager.

Features:

View tasks

Add tasks

Mark tasks as completed

Delete tasks

Exit program

Run with:

Code
python todo.py
2. Simple Calculator
Performs basic arithmetic operations.

Features:

Addition

Subtraction

Multiplication

Division (with zero check)

Run with:

Code
python calculator.py
3. Password Generator
Generates random passwords based on user input.

Features:

User specifies password length

Option to choose complexity (letters only, letters + digits, or strong with symbols)

Can generate multiple passwords in one run

Run with:

Code
python password_generator.py
4. Rock‑Paper‑Scissors Game
Classic game against the computer.

Features:

User chooses rock, paper, or scissors

Computer randomly selects

Game logic decides winner

Score tracking for multiple rounds

Option to play again
